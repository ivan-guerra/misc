\doxysubsection{Detailed Description}
Represent the player\textquotesingle{}s space ship. 

\mbox{\hyperlink{classDiceInvaders_1_1PlayerShip}{Player\+Ship}} is an object that exposes the interface needed to draw a space ship sprite in different locations over the object\textquotesingle{}s lifetime. \mbox{\hyperlink{classDiceInvaders_1_1PlayerShip}{Player\+Ship}} objects also process keyboard events which include left/right movement of the ship as well as firing of a rocket.
