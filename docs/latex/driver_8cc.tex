\doxysubsection{Detailed Description}
Execute the Super Word Search puzzle solver on a user specified puzzle file. 

This driver prompts the user for a word search puzzle data file and then uses an object of the Word\+Search\+Solver to solve all puzzles contained within the data file. All output is sent to stdout. Error messages if any are sent to stderr.
