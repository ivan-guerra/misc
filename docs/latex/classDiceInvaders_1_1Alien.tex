\doxysubsection{Detailed Description}
Represent an alien invader. 

\mbox{\hyperlink{classDiceInvaders_1_1Alien}{Alien}} is an object that exposes the interface needed to draw an alien sprite in different locations over the object\textquotesingle{}s lifetime. \mbox{\hyperlink{classDiceInvaders_1_1Alien}{Alien}} objects also periodically drop bombs and for drawing/managing the bomb over its lifetime.
