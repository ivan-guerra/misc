Detailed Description

Represent the player's space ship.

PlayerShip is an object that exposes the interface needed to draw a space ship sprite in different locations over the object's lifetime. PlayerShip objects also process keyboard events which include left/right movement of the ship as well as firing of a rocket.

void DiceInvaders::PlayerShip::kb_event_handler ( )
private

Use player keystrokes to update the state of the PlayerShip.

kb_event_handler() will update the PlayerShip position according to the user's left/right keystrokes. Additionally, if the player 'fires', kb_event_handler() will spawn a rocket object.

void DiceInvaders::PlayerShip::update ( )

Update the state of the PlayerShip in game.

update() will draw the ship in a location corresponding to the user's last move keystrokes. Additionally, update() will manage the update of this PlayerShip's rocket if any.
