Detailed Description

Declare the interface for a class whose purpose is to solve word puzzles.

WordSearchSolver exposes an interface which allows the user to (1) Load word puzzle data from a text file. (2) Solve the word puzzle in either a wrapped or no wrap mode. (3) Print the word puzzle solution to stdout. (4) Perform (1)-(3) for each subsequent puzzle in the input text file.

Load the word search grid into memory.

get_grid() attempts to read the puzzle's dimension data from the input file and then proceeds to read NxM-length strings. The strings are decomposed into chars and the end result is an NxM matrix of chars representing the word search grid. The representation is stored in word_grid_.

bool supersearch::WordSearchSolver::load_puzzle ( )

Load a puzzle from the input puzzle data file.

load_puzzle() is responsible for fetching all state data associated with a puzzle from the input file. Puzzle state data includes (1) The word search grid. (2) The search mode. (3) The list of search terms. If any of (1)-(3) fails, load_puzzle() returns false and prints an error message to stderr.

Returns

◆ solve()

bool supersearch::WordSearchSolver::solve ( )

Search for all input search terms in the word grid.

solve() attempts find search terms in the word grid by iterating over the grid and launching a directional search at each cell. This directional search goes down all 8 possible paths. If found, the search term's starting and ending indices are stored.

Returns
True if a search for each search term is performed successfully. Note, a return value of true does NOT indicate all search terms were found!

int supersearch::WordSearchSolver::wrap_move ( int  limit,

Wrap \pos around limit.

wrap_move() is a helper method of directional_search() that takes a position \pos and wraps it around a boundary, limit. There are three cases: (1) pos >= limit -> pos = 0 (2) pos < 0 -> pos = limit - 1 (3) pos does not satisfy (1)-(2) and thus remains unchanged

Returns
An integer representing the wrap of pos around limit. If no wrapping was performed, pos is returned.
