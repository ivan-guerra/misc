Detailed Description

A doubly-linked list specifically used to manage Alien objects.

The concept behind AlienList is to provide and efficient way of managing the interaction between Alien and PlayerShip objects in game. AlienList provides the interface for spawning alien and armies and updating game state based on collisions between aliens, rockets, players, and the boundaries of the game window.

void DiceInvaders::AlienList::resolve_collisions ( PlayerShip *  player,
int  direction 
)

Resolve collisions if any between player and aliens in the AlienList.

resolve_collisions() traverses the AlienList and for each alien in the list it takes actions if (1) The alien hits the player. (2) The alien's bomb hits the player. (3) The player's rocket hits the alien. The goal is to 'resolve' each of the above collision scenarios by updating player, alien, and AlienList state accordingly.

Parameters
playerA PlayerShip object.
directionThe horizontal direction of the aliens (-1:left and 1:right).
