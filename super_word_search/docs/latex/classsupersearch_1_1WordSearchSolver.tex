\doxysubsection{Detailed Description}
Declare the interface for a class whose purpose is to solve word puzzles. 

\mbox{\hyperlink{classsupersearch_1_1WordSearchSolver}{Word\+Search\+Solver}} exposes an interface which allows the user to (1) Load word puzzle data from a text file. (2) Solve the word puzzle in either a wrapped or no wrap mode. (3) Print the word puzzle solution to stdout. (4) Perform (1)-\/(3) for each subsequent puzzle in the input text file.
