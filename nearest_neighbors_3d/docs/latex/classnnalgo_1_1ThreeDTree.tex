\doxysubsection{Detailed Description}
Declare the interface for a simplified k-\/d tree where k equals 3. 

\mbox{\hyperlink{classnnalgo_1_1ThreeDTree}{Three\+D\+Tree}} exposes an interface which allows the user to\+: (1) Construct a balanced, static 3-\/d tree from of a set points in 3D space. (2) Perform radial queries (i.\+e., find all points within a specific radius of a reference point). (3) Print the inorder traversal of the tree. The tree itself is recursively defined. Operations such as tree construction and radial searches are performed recursively and therefore make heavy use of the call stack. However, the tree is guaranteed to be balanced and therefore have a depth of no more than O(logn).
