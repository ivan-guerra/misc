Detailed Description

Declare the interface for a simplified k-d tree where k equals 3.

ThreeDTree exposes an interface which allows the user to: (1) Construct a balanced, static 3-d tree from of a set points in 3D space. (2) Perform radial queries (i.e., find all points within a specific radius of a reference point). (3) Print the inorder traversal of the tree. The tree itself is recursively defined. Operations such as tree construction and radial searches are performed recursively and therefore make heavy use of the call stack. However, the tree is guaranteed to be balanced and therefore have a depth of no more than O(logn).

Find the index in coords of the median element on the current search axis.

find_median_on_axis() computes the median point by axis in the range [l, r]. The implementation has the side effect of sorting coords from index l to index r. The sort is performed with respect to the current axis (x, y, or z) which is calculated as axis = depth % 3.

Parameters
lLeft bound of coords.
rRight bound of coords.
depthLevel of the median within the tree.
coordsVector of 3D point objects.
Returns
The index of the median by axis in coords within the bounds [l, r].
