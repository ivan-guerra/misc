\doxysubsection{Detailed Description}
A doubly-\/linked list specifically used to manage \mbox{\hyperlink{classDiceInvaders_1_1Alien}{Alien}} objects. 

The concept behind \mbox{\hyperlink{classDiceInvaders_1_1AlienList}{Alien\+List}} is to provide and efficient way of managing the interaction between \mbox{\hyperlink{classDiceInvaders_1_1Alien}{Alien}} and \mbox{\hyperlink{classDiceInvaders_1_1PlayerShip}{Player\+Ship}} objects in game. \mbox{\hyperlink{classDiceInvaders_1_1AlienList}{Alien\+List}} provides the interface for spawning alien and armies and updating game state based on collisions between aliens, rockets, players, and the boundaries of the game window.
